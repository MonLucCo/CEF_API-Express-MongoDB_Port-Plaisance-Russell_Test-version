# Endpoints Catways

---

## GET /catways

Liste des catways.

---

## GET /catways/:id

Détail d’un catway.

---

## POST /catways

Créer un catway.

Payload :

```json
{
  "identifier": "A1",
  "status": "free"
}
```

---

## PUT /catways/:id

Modifier un catway.

---

## PATCH /catways/:id

Mettre à jour partiellement un catway.

---

## DELETE /catways/:id

Supprimer un catway.
