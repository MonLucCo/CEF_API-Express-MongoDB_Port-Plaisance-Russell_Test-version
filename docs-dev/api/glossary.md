# Glossaire

**Catway**  
Petit appontement permettant d’amarrer un bateau.

**Réservation**  
Blocage d’un catway pour une période donnée.

**JWT (JSON Web Token)**  
Jeton d’authentification utilisé pour sécuriser les routes.

**Endpoint**  
URL représentant une ressource de l’API.

**Payload**  
Contenu JSON envoyé dans une requête POST ou PATCH.
