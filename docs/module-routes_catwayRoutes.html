Module: routes/catwayRoutes

Définition des routes Catways de l’API Port de Plaisance Russell. Ce module regroupe l’ensemble des endpoints liés aux catways. Il intègre les middlewares : - de validation et de résolution d’identifiant (issue‑26) - de validation de payload (issue‑27) Routes disponibles : - GET /catways → liste des catways - GET /catways/:id → détail d’un catway (identifiant hybride) - POST /catways → création d’un catway - PUT /catways/:id → mise à jour complète d’un catway - PATCH /catways/:id → mise à jour partielle d’un catway - DELETE /catways/:id → suppression d’un catway Ce routeur est monté dans `app.js` sur le chemin `/catways` : app.use('/catways', catwayRoutes);
